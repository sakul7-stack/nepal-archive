# Nepal Media Archive

**Nepal Media Archive** is a project that collects and stores Nepali media content daily.

Website: https://archive.kushal-kc.com.np

## Overview

Nepal Media Archive automatically archives different types of media content from Nepal.  
The goal is to preserve publicly available media and create a historical record of news and online daily activity.

The archive collects data from:

- News e-portals
- Newspapers
- Social media pages (limited to reddit,youtube trending)

## Scraping Tools

The project supports two scraping methods:

### Selenium
works well but not optimized.

### Playwright
A more efficient.
